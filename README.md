# Job Fraud Detection App

This project helps detect fraudulent job postings using a React frontend and Node.js backend.  
- *Frontend*: Built with React, providing a user interface to enter job details.  
- *Backend*: Node.js + MongoDB, running fraud detection logic.  
- *Features*:  
  - Detects fraud based on job title, company, and description.  
  - Color-coded status (Fraud / Legit).  
  - Inline editing and smooth updates.  

## How to run locally
1. Clone the repo: git clone https://github.com/mallimalleswari217-lang/job-fraud-detection.git
2. Navigate to backend: cd backend → run npm install → npm start
3. Navigate to frontend: cd frontend → run npm install → npm start

## Deployment
- Backend deployed on Render.  
- Frontend deployed on Vercel.
